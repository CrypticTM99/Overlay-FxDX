Overlay FXDX v1.2.2 (Crafted by CrypticTM)

Overlay FXDX is a real-time visual overlay system for SM64 CoopDX that adds retro and stylized display filters directly in-game. It is designed to be lightweight, simple to use, and easy to extend.

For Modders

This project is intentionally open and flexible.

You are free to:

Modify the existing overlays
Add new visual effects
Rework the system entirely
Use it as a base for your own overlay or shader-style mods

Credit is not required, but always appreciated if you choose to use or build on it.

I’m also happy to help newer modders understand or expand on the system if needed.

Please keep any modifications appropriate and in line with community standards.

For Players

Overlay FXDX lets you change how the game looks in real time using simple visual filters.

These include effects like:

Retro CRT-style display
Black & white filter (WIP)
VHS-style distortion
Warm/Cool Tint
Buggy effects
Glitch effects
Other stylized overlays

The effects are purely visual and do not affect gameplay.

Controls
L Trigger – Cycle overlay modes

When you change an overlay, the selection menu appears briefly on screen and then fades out automatically.
